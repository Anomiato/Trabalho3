	Ao executar o programa, será pedida a data atual, que deve ser entrada na maneira dd/mm/yyyy.
	Então, é aberto o menu de opções com as funcionalidades do programa, em que deve ser entrada uma das seguintes opções:
		1 para cadastrar novos usuários no sistema. Serão pedidas as seguintes informações: nome do usuário, endereço do usuário, telefone do usuário e tipo do usuário (1 para professor, 2 para aluno e 3 para comunidade);
		2 para cadastrar novos livros. Serão pedidas as seguintes informações: título do livro, autor do livro, editora do livro, quantidade de páginas do livro e tipo de livro (1 para texto, 2 para geral);
		3 para empréstimos. Serão pedidos o nome do livro e o nome do usuário desejado. Então, é pedido que seja a entrada a data de empréstimo (Y para manter a data entrada no início e N para uma nova data);
		4 para devoluções. Serão pedidos o nome do livro, o nome do usuários e a data de devolução (a ser entrada na forma dd/mm/yyyy);
		5 para as diferentes listagens. Será pedido que seja entrada a opção de listagem (1 para listagem dos usuários, 2 para listagem de livros, 3 para listagem de empréstimos, 4 para listagem de devoluções, 5 para listagem de atrasos e 6 para voltar ao menu anterior);
		6 para sair do programa.
	Todos os dados entrados durante a execução ficam salvos nos diferentes arquivos .csv, podendo ser consultados em uma nova execução do programa.
